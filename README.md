# barsi-investing-framework
Repositório do desafio da DIO contendo um Caderno Temático criado no NotebookLM sobre a metodologia de investimentos de Luiz Barsi. Inclui contexto e objetivos de estudo, curadoria de fontes, engenharia de prompts, registros de testes e o miniguia final com resumos estruturados, glossário e prompts reutilizáveis.
